# Weather Diary

## Автор
**Алексенко Савелий**

## Описание программы
Weather Diary - это графическое приложение для ведения дневника погоды. Программа позволяет:
- Добавлять записи о погоде (дата, температура, описание, наличие осадков)
- Просматривать все записи в удобной таблице
- Фильтровать записи по дате и температуре
- Сохранять и загружать данные в формате JSON
- Проверять корректность вводимых данных

## Системные требования
- Python 3.7 или выше
- Библиотека tkinter (входит в стандартную установку Python)
